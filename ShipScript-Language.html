üíª Example Programs

1. Calculate Freight Cost

~ Calculate freight cost for a shipment CARGO calculateFreight(weight, route) { VESSEL base_rate = 280000 VESSEL total_cost ROUTE IF route == "CanTho-CaiMep" { total_cost = weight * base_rate } ROUTE ELSE { total_cost = weight * 1340000 } DELIVER total_cost } ~ Execute the cargo operation VESSEL shipment = calculateFreight(50, "CanTho-CaiMep") SIGNAL "Total cost: " + shipment + " VND"

2. Track Vessel Schedule

~ Fleet management system FLEET Vessel { OPERATION initialize(name, capacity) { this.name = name this.capacity = capacity this.trips = 0 } OPERATION sail() { this.trips = this.trips + 1 SIGNAL this.name + " departing for trip #" + this.trips } } ~ Create our fleet VESSEL alize = NEW Vessel("Aliz√© I", 600) VESSEL aquaterra = NEW Vessel("Aquaterra I", 12300) ~ Schedule 15 trips per month DOCK WHILE alize.trips < 15 { alize.sail() }

3. Port Route Optimization

~ Optimize route between ports CARGO optimizeRoute(ports) { CONTAINER optimized = [] VESSEL total_distance = 0 DOCK FOR port IN ports { ROUTE IF port.status == "OPEN" { optimized.LOAD(port) total_distance = total_distance + port.distance SIGNAL "Added port: " + port.name } } MANIFEST result = { route: optimized, distance: total_distance, estimated_time: total_distance / 25 } DELIVER result } ~ Define our ports CONTAINER ports = [ { name: "Can Tho", status: "OPEN", distance: 0 }, { name: "Cai Mep", status: "OPEN", distance: 180 }, { name: "Shenzhen", status: "OPEN", distance: 850 } ] VESSEL best_route = optimizeRoute(ports) SIGNAL "Optimized route distance: " + best_route.distance + " km"

4. Cargo Consolidation System

~ Consolidate LCL shipments CARGO consolidateCargo(shipments) { VESSEL total_weight = 0 VESSEL total_volume = 0 ANCHOR MAX_CAPACITY = 600 NAVIGATE { DOCK FOR shipment IN shipments { ROUTE IF (total_weight + shipment.weight) > MAX_CAPACITY { SIGNAL "‚ö†Ô∏è Capacity exceeded! Starting new container..." BREAK } total_weight = total_weight + shipment.weight total_volume = total_volume + shipment.volume SIGNAL "‚úì Loaded: " + shipment.id } } SALVAGE error { SIGNAL "‚ùå Error during consolidation: " + error } DELIVER { total_weight: total_weight, total_volume: total_volume, utilization: (total_weight / MAX_CAPACITY) * 100 } }
